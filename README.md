# Live Translate

**v1.2.1-beta** · Speech → streaming ASR → streaming translation on Android · [MIT](LICENSE)

**English** · [中文](README.zh-CN.md)

[![Release](https://img.shields.io/github/v/release/y12rf/live-caption-translate)](https://github.com/y12rf/live-caption-translate/releases)

Bring your own OpenAI-compatible **ASR** and **LLM**. Built for lectures and notes: live bilingual text, floating captions, history with recordings.

## Features

- Mic / internal audio / file import
- Streaming recognition + translation (or ASR-only)
- Floating captions, pause/resume, searchable history
- Glossary, reprocess past sessions, crash recovery

## Quick start

1. Install the arm64 APK from [Releases](https://github.com/y12rf/live-caption-translate/releases).
2. **Settings** → set ASR + LLM URL, key, model, auth.
3. **Live** → pick source → **Start**.

More setup detail: [doc/USAGE.md](doc/USAGE.md)
