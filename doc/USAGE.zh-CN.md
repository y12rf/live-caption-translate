# 使用与配置

安装概览见 [README](../README.zh-CN.md)。本页写配置细节。

## 配置接口

密钥只保存在**本机**。打开 **设置**，至少填写：

| 配置项 | 说明 |
|--------|------|
| ASR 地址 / Key / 模型 | 语音识别服务 |
| ASR 接口风格 | OpenAI transcriptions，或兼容的 chat+audio |
| ASR 鉴权 | `Bearer` 或 `Api-Key` |
| LLM 地址 / Key / 模型 | 兼容 Chat Completions 的翻译模型 |
| LLM 鉴权 | 一般用 `Bearer` |
| 输入 / 输出语言 | 识别语言与翻译目标（界面语言单独设置） |

### 提示

- **完整 URL**：按原样使用，不再拼接 `/v1/...`（适合自定义网关）。
- **提示词**占位符：`{{from}}`、`{{to}}`、`{{text}}`、`{{history}}`、`{{glossary}}`。
- **术语表**：词对注入 `{{glossary}}`。
- **VAD**：课堂场景默认即可；可调静音 / 最长句长。
- **仅识别**：只要转写、不要翻译时打开。
- **悬浮窗**：尺寸、颜色、透明度、显示模式；位置锁定见录音通知。

## 权限

| 权限 | 用途 |
|------|------|
| 麦克风 | 实时采集 |
| 通知 | 录音状态（Android 13+） |
| 悬浮窗 | 悬浮字幕 |
| 媒体投屏 | 内录 |

## 日常使用

| 操作 | 说明 |
|------|------|
| 开始 / 暂停 / 停止 | 实时页；停止会保存历史与 WAV |
| 悬浮字幕 | 实时页开关；通知栏锁定 |
| 全屏双语 | 顶栏全屏图标 |
| 回放 / 重跑 / 导出 | 历史页 |

## 已知限制

- 延迟 ≈ 切句长度 + ASR/LLM 网络。
- 内录质量因厂商而异。
- API Key 明文存本机；局域网网关允许 cleartext HTTP。
- 数据库 schema 升级可能清空历史。
- FFmpeg 会增大 APK。
