# Live Translate（实时翻译）

**v1.2.0** · 语音 → 流式识别 → 流式翻译 · [MIT](LICENSE)

[English](README.md) · **中文**

[![Release](https://img.shields.io/github/v/release/y12rf/live-caption-translate)](https://github.com/y12rf/live-caption-translate/releases)

自备 OpenAI 兼容的 **ASR** 与 **LLM**。适合上课记笔记：实时双语、悬浮字幕、带录音的历史记录。

## 功能

- 麦克风 / 内录 / 文件导入
- 流式识别 + 翻译（也可仅识别）
- 悬浮字幕、暂停继续、可搜索历史
- 术语表、历史重跑、异常恢复

## 快速开始

1. 从 [Releases](https://github.com/y12rf/live-caption-translate/releases) 安装 arm64 APK。
2. **设置** → 填写 ASR / LLM 的地址、Key、模型、鉴权。
3. **实时** → 选音源 → **开始**。
